## Microphone Issue Investigation

- User reported "Microphone access denied" on Control UI via `claw.kilosessions.ai`
- Root cause: `Permissions-Policy: camera=(), microphone=(), geolocation=()` header hardcoded in OpenClaw's static file server — no config option to override
- Attempted Cloudflare Worker to override header — Worker deployed and working, but route to `claw.kilosessions.ai/*` didn't intercept (domain is in a different Cloudflare account that Richard doesn't control)
- Workaround: Patched `Cu()` function in `index-UvgeZ3yV.js` to `return false`, hiding the mic button entirely. Backup at `.bak`. Will be overwritten on OpenClaw updates.
- Also discovered: Control UI file attachments silently fail (show thumbnail but never deliver) — separate bug
- GitHub issue drafts saved to `github-issues-drafts.md` (both issues)
- OpenClaw update available: `npm 2026.3.24`
