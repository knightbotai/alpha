### WebRTC Prototype Research (2026-04-04)

**FastRTC Library:**
- Python WebRTC library (gradio-app/fastrtc)
- Built-in VAD (voice activity detection) for turn-taking
- Can mount on FastAPI or use built-in Gradio UI
- Has example code for: Whisper STT → LLM → TTS pipeline
- Install: `pip install "fastrtc[vad, tts]"`

**Architecture:**
```
Microphone → FastRTC → Whisper STT → vLLM → Kokoro TTS → FastRTC → Speaker
```
